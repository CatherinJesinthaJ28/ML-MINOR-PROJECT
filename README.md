# Apple Stock Price Prediction

## Machine Learning Minor Project

This project predicts Apple's closing stock price using Linear Regression.

### Features
- Dataset Preview
- Data Visualization
- Train/Test Split
- Linear Regression Model
- Stock Price Prediction
- Accuracy Score

### Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
